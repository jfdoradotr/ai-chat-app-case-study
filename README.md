# AI Chat App — Case Study

Case study project built as part of the [SwiftUI Advanced Architecture](https://www.swiftful-thinking.com/offers/REyNLwwH) course by [Swiftful Thinking](https://www.swiftful-thinking.com). This project covers **Part 1: AI Chat App** of the course.

## About the Course

**SwiftUI Advanced Architecture** is a 65+ hour course focused on building enterprise-ready, full-stack SwiftUI applications. The course is divided into three parts:

1. **AI Chat App** — Build a fully functional AI-powered chat application (this project)
2. **MVP Essentials** — Core patterns for shipping a minimum viable product
3. **Enterprise Architecture** — Advanced architectural patterns for production apps

## Objectives

- [ ] Build an AI chat application using SwiftUI
- [ ] Apply advanced architecture patterns learned throughout the course
- [ ] Document key learnings and decisions
